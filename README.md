## Technology stack

- Kotlin
- Multi module
- Clean Architecture (Jetpack ViewModel + Flow)
- Jetpack Compose
- Work Manager
- Hilt
- Coroutines
- Retrofit
- Okhttp
- Kotlin Serialization
- Compose Navigation
- Firebase Cloud Messaging
- Data Store
- Jetpack Paging 3
- Coil
- Exo Player
